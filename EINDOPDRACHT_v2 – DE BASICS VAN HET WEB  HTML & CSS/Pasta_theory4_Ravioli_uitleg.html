Ravioli

Ravioli code bestaat uit object georienteerde programmering opgebouwd uit goed gestuctueerde classes welke opzichzelf staand goed te begrijpen zijn. In ravioli code zijn alle componenten en objecten opzichzelf staand waardoor deze allemaal gewijzigd moeten kunnen worden zonder (onbedoeld) de programmering achter andere componenten en objecten mee te wijzingen.
