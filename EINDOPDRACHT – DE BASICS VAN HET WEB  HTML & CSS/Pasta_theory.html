Pasta theorie

De spaghetti theorie staat niet opzichzelf, onsamenhangende code is er in verschillende vormen en in verschillende maten. Ook bestaat er nog zoiets als een gewenste wijze van coderen, welke een logische en overzichtelijke samenhang in de codestructuur biedt, waardoor deze ook voor anderen leesbaar en begrijpelijk is. Spaghetti code staat dus niet geheel opzichzelf, hier is nog heel Italiaans deegwaar repertoire aan metaforen aan toe te voegen:

Spaghetti

Spaghetti code wordt als metafoor gebruikt voor ongestructueerdere en moeilijk te onderhouden source code. Net zoals een bord spaghetti ziet spaghetti code eruit als een verwarde brei, ingewikkeld om te volgen, en zeker voor anderen moeilijk te begrijpen. Klik hier voor bron.

Lasagne

Lasagne code bevat net zoals lasagne een zekere gestructureerde laag, kan als leesbaar beschouwd worden maar door zijn monolitische aard zijn de structuren van de code zo ingwikkeld met elkaar verweven dat het risico als nog groot is dat de code (voor anderen) onleesbaar wordt. Een voorbeeld hiervan is een object georienteerde code waarin veel kleine classes en meerdere onvoorspelbare onderlinge afhankelijkheden worden gerbruikt. Klik hier voor bron.

Ravioli

Ravioli code bestaat uit object georienteerde programmering opgebouwd uit goed gestuctueerde classes welke opzichzelf staand goed te begrijpen zijn. In ravioli code zijn alle componenten en objecten opzichzelf staand waardoor deze allemaal gewijzigd moeten kunnen worden zonder (onbedoeld) de programmering achter andere componenten en objecten mee te wijzingen.Klik hier voor bron.

Pizza

Pizza code lijkt op lasanga code maar beschikt niet over een lagenstructuur en is dus architectonisch plat te noemen. De grote brei aan classes vormen een plat onsamenhangend geheel. Net zoals de toppings op een pizza. Klik hier voor bron.

spaghetti di chaotti

Kun je het al ruiken? Die heerlijk baal vol dampende spaghetti met rode saus en een zooi kaas? Zie je het voor je? Die zelfbereide, op je bord gekwakte, chaotische brei? Ik wel! En ik geniet van elke hap, en van elke stap, tot ik op een dag klaar ben om je een bord met gestrutureerde ravioli voor te schotelen.
